筆記

使用leo editer編輯,

然後利用github推送,

步驟:

cd 某資料夾

git clone 倉儲網址

git branch=查看分支

git branch gh-pages=建立分支

git checkout gh-pages=轉換分支

git add -A=加入檔案至索引

git commit -m "倉儲"=複製檔案至倉儲

git push origin "分支"=推送檔案至分支
